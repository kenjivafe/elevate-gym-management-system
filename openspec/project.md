# Project Context

## Purpose
A gym membership card system that allows gym owners to manage their members and generate membership cards and the cards should contain the details of the member using rfid technology. The cards will also be used for an RFID locker. The system also have a dashboard for gym owners to manage their members and generate membership cards. The system also have a employee attendance system we will implement biometrics for this.

## Tech Stack
- Next.js
- TypeScript
- Tailwind CSS

## Project Conventions

### Code Style
Clean with proper documentation and comments, following best practices.

### Architecture Patterns
Clean architecture with separation of concerns.

### Testing Strategy
Unit tests for critical business logic, integration tests for API endpoints.

### Git Workflow
Feature branches for development, main branch for production.

## Domain Context
Gym management, membership card generation, RFID technology, employee attendance with biometrics.

## Important Constraints
- Must support real-time RFID card generation
- Must support employee attendance with biometrics
- Must support gym management dashboard
- Must support RFID locker integration

## External Dependencies
- RFID card generation API
- Employee attendance biometric API
- Gym management dashboard API
- RFID locker integration API
